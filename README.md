# Moyamoya
This repository contains ML models to predict postoperative cerebral infarction in patients with Moyamoya disease.

Python scripts of the machine learning algorithms. The current work is in "Development and validation of machine learning models to predict postoperative infarction in moyamoya disease."

'Internal_validation.py': Comparison of the three machine learning models in the internal validation.
'External_validation.py': Machine learning models in the external validation and the best performing model's SHAP.
